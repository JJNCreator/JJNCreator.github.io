Legends Reborn: Age of Chance

Legends Reborn: Age of Chance is an NFT-based collectible card game made by Kung Fu Factory during my time at Max Gaming Studios.



Contributions

Keybind management, venue management, store and gachas feature, emote implementation, campaign/league implementation, documentation assistance, bug fixing

Software used

Unity, Visual Studio, JetBrains Rider, Meld, Perforce, Photoshop

Technical challenges

Some issues I ran into were getting the UI to align with the mockups. This resulted in several iterations of the menus I worked on. I also ran into some issues with setting up the analytics.

Lessons learned

Communication is incredibly important, especially in remote work. One thing I learned was that with Perforce, you need to make sure another team member isn't already looking at the file you want to edit. This caused a couple sync issues during my early days on the project.
